Purpose

ProteinEvolverABC estimates recombination and substitution rates from alignments of protein sequences by approximate Bayesian computation (ABC). The second version of the framework allows for the estimation of these parameters accounting for evolutionary constraints from the protein structure.

The file ProteinEvolverABC2.0.manual.pdf includes detailed documentation, and the user is strongly advised to read it carefully before using the program.


Download

Download the program from https://github.com/MiguelArenas/proteinevolverabc/. Open a console window. In the same directory as the file ProteinEvolverABC2.0.zip, type:

unzip ProteinEvolverABC2.0.zip

Note that this zip file is built on a macintosh, so in other OS you might see some strange files, that start with "." or "-". These are macintosh hidden files, which can be safely removed without affecting the software.


Compilation

The program should compile without much problem in Unix-like environments (i.e., Linux). Since ABC is based on numerous simulations, this framework is particularly recommended for running on supercomputers or computer clusters. A makefile is provided for the gcc compiler, but if you have cc instead, just change gcc for cc in the makefile. The default optimization level is set to -O3. To compile it on intel processors, and probably in other machines, you can explore the option -fast instead.

To compile the program on the command-line Unix-like window (i.e., Linux), move into ProteinEvolverABC2.0.0 folder and type:

make all


ProteinEvolverABC also requires the installation of Perl and R (with the libraries: "lattice", "MCMCpack", "ape", "graphics" and "abc").

Folders "source" and "scripts" must not be modified and should be present in the provided directory, as they include fundamental tools for the analysis.


Execution

The program runs on the command-line Unix-like window.

The execution consists of 5 consecutive steps. To run it move to the folder where the main files are located and type:


perl ProteinEvolverABC_Phase1.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase2.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase3.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase4.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase5.pl PATH/Settings.txt


Where "Settings.txt" is the main input file with the user-specified settings placed in the directory. PATH

The execution of "ProteinEvolverABC_Phase3.pl" (simulation of multiple alignments of protein sequences) can, optionally, run in parallel on a user-specified number of processors.


REMEMBER:

- The structure of the package ProteinEvolverABC must not be changed, particularly, the folders "source" and "scripts"

- ProteinEvolverABC must run in the folder where the main files are located

- The target sequences alignment file (in case of using structural contraints, as well as the files of the protein structure and the sequence of that protein structure) must be located in the folder of the "Settings.txt" file.

- Read the software documentation before using the program

- Several practical examples, including input and output files, are provided to facilitate the use of the framework


To remove the executable files on the command-line Unix-like window (i.e., Linux), move into ProteinEvolverABC2.0.0 folder and type:

make clean
