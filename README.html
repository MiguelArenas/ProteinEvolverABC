Purpose

ProteinEvolverABC estimates recombination and substitution rates from alignments of protein sequences by approximate Bayesian computation (ABC).

The file ProteinEvolverABC1.0.manual.pdf includes detailed documentation, and the user is strongly advised to read it carefully before using the program.


Download

Download the program from https://github.com/MiguelArenas/proteinevolverabc/. Open a console window. In the same directory as the file ProteinEvolverABC1.0.zip, type:

unzip ProteinEvolverABC1.0.zip

Note that this zip file is built on a macintosh, so in other OS you might see some strange files, that start with "." or "-". These are macintosh hidden files, which can be safely removed without affecting the software.


Compilation

The program should compile without much problem in any Unix-like environment (Linux, MacOSX, etc.). A makefile is provided for the gcc compiler, but if you have cc instead, just change gcc for cc in the makefile. The default optimization level is set to -O3. To compile in macintosh with an intel processor, and probably in othe machines, you can enable the option -fast instead.

To compile the program move into ProteinEvolverABC2.0.0 folder and type:

make all


ProteinEvolverABC also requires the installation of Perl and R (with the libraries: "lattice", "MCMCpack", "ape", "graphics" and "abc")

Folders "source" and "scripts" must not be modified


Execution

The program runs on the command-line Unix-like window. In MacOS X or Linux the user needs to open the Terminal application.

The execution consists of 4 consecutive steps. To run it move to the folder where the main files are located and type:


perl ProteinEvolverABC_Phase1.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase2.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase3.pl PATH/Settings.txt


perl ProteinEvolverABC_Phase4.pl PATH/Settings.txt


Where "Settings.txt" is the main input file with the user-specified settings placed in the directory PATH

The execution of "ProteinEvolverABC_Phase2.pl" (simulation of multiple alignments of protein sequences) can, optionally, run in parallel under a user-specified number of processors


REMEMBER:

- The structure of the package ProteinEvolverABC must not be changed, particularly, folders "source" and "scripts"

- ProteinEvolverABC must run from the folder where the main files are located

- The target sequences alignment file must be located in the folder of the "Settings.txt" file

- Read the software documentation before using the program
