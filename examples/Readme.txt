To analyze these examples:
After compiling ProteinEvolverABC (if required).

1. Copy the input files of the examples (Settings.txt and multiple sequence alignment, and if required the files with the protein structure (.pdb) and the sequence of the protein structure (seqGMRCA)) in the working directory (where ProteinEvolverABC_Phase executable files and folders “bin” and “scripts” are located).

2. Run ProteinEvolverABC in the working directory.
